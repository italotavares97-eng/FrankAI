Fluxo

Como funciona

Do dado bruto ao relatório executivo em segundos.

1
CONECTORES

Coleta de dados em paralelo

Sults ERP (vendas + estoque), Meta Graph API (ads performance), LinkedIn (pipeline B2B), WhatsApp Z-API — tudo via asyncio.gather().

2
SWARM

9 agentes analisam em paralelo

CEO chama todos os diretores simultaneamente. Cada agente usa cache-aside Redis (TTL 30min) para evitar chamadas redundantes ao LLM.

3
CEO RULES

Validação das regras hard do CEO

AgentContext.check_ceo_rules() valida CMV, EBITDA, aluguel, payback e ROI em cada setor. Violações geram alertas imediatos.

4
SÍNTESE

Relatório executivo via Claude 3.5 Sonnet

CEO Agent sintetiza todos os resultados setoriais num relatório executivo com prioridades, ações e recomendações.

5
NOTIFICAÇÃO

Entrega automática por Email + WhatsApp

Briefing matinal, alertas críticos e relatórios semanais/mensais entregues automaticamente. Ações como pausar campanhas executadas sem intervenção humana.
